# VendorBridge AI — Your AI Procurement Officer

> **AI-powered procurement platform that reduces RFQ-to-PO cycle time from 45+ days to 7 days**

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- Supabase account
- Google Gemini API key
- Resend API key

### Installation

```bash
git clone https://github.com/1hetal123/vendorbridge-ai.git
cd vendorbridge-ai
npm install
```

### Environment Setup

Create `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
GEMINI_API_KEY=your_gemini_api_key
RESEND_API_KEY=your_resend_api_key
NEXT_PUBLIC_APP_URL=http://localhost:3000
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
